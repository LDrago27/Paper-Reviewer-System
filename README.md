# Paper-Reviewer-System
The paper Reviewer System(TCRSys) is a novel multi layered approach to solve the problem of paper reviewer assignment. 
The System consists of 3 Important layers Topic Network, Citation Network and Reviewer Network.
The folders contains 3 notebooks outlying the construction of 2 layers.
Apart from that the files of Main And Main_Baseline is usedd for testing purposes.
All files except 2 files that were more than 100 mb are absent.
